## üéÆ What is GDevelop?

GDevelop is a **full-featured, no-code, open-source** game engine that allows users to create:

- ‚úÖ **2D Games** - Platformers, shooters, puzzles, RPGs
- ‚úÖ **3D Games** - 3D platformers, racing, first-person games
- ‚úÖ **Multiplayer Games** - Real-time multiplayer experiences
- ‚úÖ **Mobile Games** - iOS and Android support
- ‚úÖ **Web Games** - Export to HTML5

### Key Features

- **No-Code**: Visual event-based system (no programming required!)
- **Visual Editor**: Drag-and-drop game creation
- **Behaviors**: Reusable game behaviors
- **Physics**: Built-in physics engine
- **Export**: Multiple export options (web, mobile, desktop)
